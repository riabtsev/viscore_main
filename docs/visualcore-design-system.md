# VisualCore — Design System (Главная)

Документ фиксирует визуальное ядро главной страницы VisualCore: базовые токены, принципы композиции, типографику, UI-паттерны и правила анимации.

## 1) Visual Core

### Бренд-идея
- **Образ:** образовательный продукт на стыке геймификации и современного editorial UI.
- **Контраст:** тёплая светлая основа + тёмные карточки + яркий акцент.
- **Эмоция:** «дружелюбно, технологично, живо».

### Визуальная формула
- **База:** мягкий бежевый фон (`#F5F0EB`) как спокойный холст.
- **Опора:** угольно-тёмный (`#111110`, `#171614`, `#1A1917`) для глубины и контраста.
- **Энергия:** акцентный оранжевый (`#FF4F1F`) для CTA, прогресса и ключевых метрик.
- **Сигналы:** функциональные цветовые коды для статусов и модулей (синий, зелёный, янтарный, красный).

### Типографика
- **Семейство:** `Satoshi`, fallback `system-ui, sans-serif`.
- **Веса:** `400`, `500`, `700`, `900`.
- **Тональность:** плотные жирные заголовки + мягкий читаемый body.

#### Типовая шкала (фактический стиль главной)
- **Display / Hero H1:** `clamp(3rem, 7vw, 6.25rem)`, `font-weight: 900`, `line-height: 0.88`.
- **Section H2:** `clamp(2rem, 4.5vw, 3rem)`, `font-weight: 900`.
- **Card H3:** ~`1.25rem`, `font-weight: 900`.
- **Body L:** `1rem–1.125rem`, `line-height: ~1.6`.
- **Body S:** `0.875rem`.
- **Meta / Label:** `0.625rem–0.75rem`, uppercase + tracking.

### Радиусы
- **Мелкие:** `rounded-lg` / `rounded-xl` (`~12px`) для микрокомпонентов.
- **Основные:** `rounded-2xl` (`~16px`) для карточек и кнопок.
- **Крупные:** `rounded-3xl` (`~24px`) для hero/bento блоков.
- **Pill:** полный радиус для бейджей и статусных меток.

### Тени и свечение
- Мягкие тени для светлых surface (`shadow-sm`).
- Глубокие тени в dark-контексте.
- Локальные цветные glow-подсветки у ключевых чисел и CTA (обычно от primary).

### Сетка и брейкпоинты
- Контейнер: `max-w-7xl`, горизонтальные поля `px-4 sm:px-6 lg:px-8`.
- Сетка секций: mobile-first, переход к `md`/`lg`.
- Доп. брейкпоинты в теме:
  - `2xsm: 375px`
  - `xsm: 425px`
  - `3xl: 2000px`

## 3) Композиция главной

### Структура страницы
1. `NavBar` (фиксированный, полупрозрачный фон с blur).
2. `Hero`.
3. Marquee-лента инструментов.
4. Stats bar.
5. Секция модулей.
6. How it works (тёмная).
7. About/CTA (bento-композиция).
8. `Footer`.

### Принципы композиции
- Чередование **светлых и тёмных секций** для ритма.
- Микс **строгой сетки** и **асимметричных акцентных карточек**.
- Высокая плотность визуальных артефактов компенсируется:
  - крупными полями;
  - явной иерархией (заголовок → описание → действие);
  - повторяемыми паттернами карточек.

## 4) Компонентные паттерны (Visual Core UI)

### Кнопки
- **Primary CTA:** оранжевый фон, белый текст, жирный вес, мягкая тень, `rounded-2xl`.
- **Secondary CTA:** светлый фон/контур, тёмный текст.
- **Motion:** scale при hover/tap; shimmer для premium-action (ограниченно).

### Карточки
- Базовый паттерн: контрастный surface + мягкий бордер + крупный радиус.
- На dark-секциях: бордеры низкого контраста (`~#252320`) и muted-текст.
- Слои: контент + декоративный фон + интерактивный слой (hover/animation).

### Бейджи/чипы
- Используются как:
  - статус прохождения;
  - уровни/XP;
  - маркеры категорий.
- Важное правило: у статуса всегда есть цветовой смысл (не только форма).

### Данные и прогресс
- Number ticker для «живых» метрик.
- Линейный progress bar и circular progress ring.
- Карточки метрик: число крупно, подписи компактно, вторичный текст приглушён.

### Навигация
- Десктоп: минималистичная строка ссылок + 2 действия справа.
- Мобайл: off-canvas через `Dialog`.
- Состояния: hover/active/focus должны оставаться контрастными на белом фоне.

## 5) Motion System

### Общие правила
- Анимация поддерживает восприятие, а не отвлекает.
- Основной характер: мягкий ease-out, умеренная длительность.
- Часто используемые приёмы:
  - `fade + y + blur` для входа блоков;
  - scale/hover для интерактивных элементов;
  - циклические микродвижения (бейджи, плавающие чипы);
  - прогрессивная отрисовка прогресса.

### Примеры таймингов
- Вход секций/карточек: `~0.4–0.6s`.
- Индикаторы прогресса: `~1.1–1.5s`.
- Бесконечные циклы: `~2.4–3.4s` (невысокая амплитуда).

### Reduced Motion
- Для чувствительных эффектов предусмотрено отключение (`prefers-reduced-motion`).
- Принцип: сохранение смысла интерфейса даже без motion.

## 6) Иллюстративный язык

- Геометрический визуальный словарь: точки, линии, формы, сетки.
- «Учебные» мини-превью встроены в карточки (не абстракция ради декора).
- Арт-направление: сочетание «продуктовой UI-карточки» и «редакционного постера».

## 7) Tone of UI Copy

- Короткие, энергичные, понятные формулировки.
- Язык действия: «Начать», «Проверить», «Открыть», «Получить XP».
- Акцент на прогрессе и результате пользователя.

## 8) Доступность и качество

- Фокус на контрасте текста к фону, особенно на тёмных секциях.
- Интерактивные элементы имеют предсказуемый hover/focus.
- Навигационные и социальные ссылки снабжены `aria-label`.
- Типографика и отступы поддерживают читаемость на мобильных.

## 9) Правила расширения системы

1. Новый экран сначала собирается из существующих паттернов карточек/CTA/метрик.
2. Новый цвет добавляется только при чёткой семантической роли.
3. Любая декоративная анимация проходит проверку:
   - помогает ли она восприятию?
   - не мешает ли чтению?
4. Если блок становится визуально «громким», сначала снижать контраст/насыщенность, а не убирать структуру.
5. Сохранять ключевую ДНК Visual Core: **тёплая база + тёмная глубина + оранжевый импульс + геометрическая обучающая метафора**.
